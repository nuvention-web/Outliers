# Keep it Personal (KIP)

All you recomindations in one place! 

## Running Locally

Make sure you have [Node.js](http://nodejs.org/) 

```sh
$ cd kip/kip_node_js
$ npm install
$ node app.js
```

Your app should now be running on [localhost:5000](http://localhost:5000/).

## Deploying to Heroku

```
$ heroku create
$ git push heroku master
$ heroku open
```
